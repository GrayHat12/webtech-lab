HTML INTRO

HTML stands for Hyper Text Markup Language. It is used to design web pages using markup language. HTML is the combination of Hypertext and Markup Language. Hypertext defines the link between the web pages. Markup language is used to combine the text document within tag which defines the structure of web pages.
